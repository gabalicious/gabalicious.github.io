Ruby Hashes and Arrays

Posted on: 11/16/2014 Read Time: 5 Minutes

Arrays

An Array is represented by square brackets [ ]. Below is some sample code for an example of an array in ruby. The array index starts at 0 so to print the first item to the console one would type puts array1[0]. To print the second item in the array one would type "puts array1[1]" and so on for the next sequential item in the array.

array1 = ["hello","world","!!!"]

Hashes

A hash is represented by squigly brackets { }. Below is some sample code for an example of a hash in ruby. Each city is represented by a symbol or abbreviation for the city it represents. Example, :bk is the symbol for the city "Brooklyn". To print each item in the hash you put the name of the hash followed by a square bracket with the key inside as show to the right. location[:bk].

location = { :bk => "Brooklyn", :si => "Staten Island", :ma => "Manhatten", :bx => "The Bronx", :qu => "Queens" }

Similarities between Arrays and Hashes.

Both Arrays and Hashes are considered indexed collections. Both arrays and hashes store a series of objects like strings, numbers, or other arrays using a key. Also both, can expand to hold more objects.
