Ruby classes vs. JavaScript constructor functions

Posted on: 10/10/2014 Read Time: 5 Minutes

Ruby and JavaScript have different ideas about classes. Let's see examples below how they are similar and different.

Ruby Class

class Player

    attr_accessor :height, :dob, :stats, :picture, :rating, :name

    def initialize(name,height,dob,rating)
        @name = name
        @height = height
        @dob = dob
        @stats = {}
        @rating = rating
    end

    def printInfo
        puts "name: #{@name}"
        puts "height: #{@height}"
        puts "dob: #{@dob}"
        puts "rating: #{@rating}"
    end

end

The code above is an example of a Ruby class. I initialized some variables and then created a printInfo Method. Creating the Object in Ruby

require_relative 'classes.rb'
p1 = Player.new("Rubin TI Jones","5'11","01-13-1989","101") # (name,height,dob,rating)
p1.rating ="105"
p1.printInfo

In the Code above I include the Class and create a new instance of the object stored in p1. Then I call the printInfo function using p1.printInfo.

name: Rubin TI
height: 5'11
dob: 01-13-1989
rating: 101

JavaScript constructor functions

 player = function(name, height, dob, rating) {
    this.name = name,
    this.height = height,
    this.dob = dob,
    this.rating = rating,
    this.printInfo = function() {
        console.log("name: " + this.name);
        console.log("height: " + this.height);
        console.log("dob: " + this.dob);
        console.log("rating: " + this.rating);
        };
 };

You can see that to define a class in JavaScript it's a bit different that in Ruby. We have to use a constructor function so that we can initialize the new object with the name, height, dob, and rating parameters. You can also see that instead of using @ next to each variable we use this.

Creating the Object with JavaScript

require('./t7-constructor_functions.js');
var p1 = new player("Rubin TI Jones", "5'11", "01-13-1989", "101");
p1.rating = "105"
p1.printInfo();

You can see that first we create a new object p1 and make it a new instance of player using var p1 = new player(parameters);. Please note you need unhide the require line to get this to work on your machine. Then after we need to call the printInfo function on p1 we must write p1.printInfo() with parenthesis in order to properly call the function. As you can see below you get the same output to ther terminal.

You can see that first we create a new object called p1 and which is a new instance of Player.

var p1 = new player(parameters);

Now we have access to call the printInfo method on p1 we like so.

p1.printInfo();
